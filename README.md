# screenshot-preventer

 

## Installation

```sh
npm install screenshot-preventer
```

## Usage


```js
import { enable, disable } from 'screenshot-preventer';

// ...

enable(); // to prevent screenshots and recordings
disable(); // to allow screenshots and recordings
```
